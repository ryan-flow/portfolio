## 🛠️ 技术栈

- **框架**: React 18 + TypeScript + Vite
- **样式**: Tailwind CSS + MUI (Material-UI)
- **路由**: React Router v6
- **部署**: GitHub Pages + GitHub Actions

## 🚀 本地开发

```bash
git clone https://github.com/oldking-yes/portfolio.git
cd portfolio
npm install
npm run dev
```

## 📦 构建部署

```bash
npm run build
npm run preview  # 本地预览生产版本
```

推送 main 分支后，GitHub Pages 自动部署。
